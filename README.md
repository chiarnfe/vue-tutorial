## 建議先安裝：Markdown Preview Enhanced

為了在 VS Code 中更方便預覽這些 Markdown 教學文件，建議先安裝 **Markdown Preview Enhanced** 擴充套件：

1. 在 VS Code 左側點選「Extensions」圖示（或按 `Ctrl + Shift + X`）。
2. 搜尋關鍵字：`Markdown Preview Enhanced`。
3. 安裝由 *Yiyi Wang* 發佈的套件。
4. 開啟任一 `.md` 檔案後，按右鍵選擇「Markdown Preview Enhanced: Open Preview to the Side」，即可在右側即時預覽。

官方連結：<https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced>
