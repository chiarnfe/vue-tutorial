# Tailwind CSS 3.4.7 切版常用 class 速查：Flex 與 Grid

> 本文件以 Tailwind CSS v3.4.7 為基準，整理常用切版（layout）相關 class，重點在 **Flex** 與 **Grid**，輔以少量實用輔助 class。

---

## 1. 基本觀念與響應式前綴

- 響應式前綴：`sm:`、`md:`、`lg:`、`xl:`、`2xl:`
  - 範例：`md:flex`（在 md 以上螢幕才套用 flex）
- 狀態前綴：`hover:`、`focus:` 等
- 方向前綴：`rtl:`、`ltr:`（較少用在切版，僅略述）

```html
<div class="flex flex-col md:flex-row">
  <!-- 手機顯示直排，md 以上橫排 -->
</div>
```

---

## 2. Flex 版面（Flexbox）

### 2.1 啟用 Flex

- `flex`：`display: flex;`
- `inline-flex`：`display: inline-flex;`

```html
<div class="flex">...</div>
<div class="inline-flex">...</div>
```

### 2.2 主軸方向（flex-direction）

- `flex-row`：水平方向（預設，從左到右）
- `flex-row-reverse`
- `flex-col`：垂直方向（從上到下）
- `flex-col-reverse`

```html
<div class="flex flex-row">...</div>
<div class="flex flex-col md:flex-row">...</div>
```

### 2.3 是否換行（flex-wrap）

- `flex-nowrap`：不換行（預設）
- `flex-wrap`：自動換行
- `flex-wrap-reverse`

```html
<ul class="flex flex-wrap gap-2">
  <li class="px-2 py-1 bg-slate-100">Tag 1</li>
  <li class="px-2 py-1 bg-slate-100">Tag 2</li>
  <!-- ... -->
</ul>
```

### 2.4 主軸對齊（justify-content）

| Tailwind class    | 對應 CSS                          |
| ----------------- | --------------------------------- |
| `justify-start`   | `justify-content: flex-start;`    |
| `justify-end`     | `justify-content: flex-end;`      |
| `justify-center`  | `justify-content: center;`        |
| `justify-between` | `justify-content: space-between;` |
| `justify-around`  | `justify-content: space-around;`  |
| `justify-evenly`  | `justify-content: space-evenly;`  |

```html
<nav class="flex items-center justify-between">
  <div>Logo</div>
  <div class="flex gap-4">Menu</div>
</nav>
```

### 2.5 交錯軸對齊（align-items / align-content / align-self）

**整排項目（align-items）：**

- `items-start` / `items-end`
- `items-center`
- `items-baseline`
- `items-stretch`（預設）

**多行內容（align-content）：**（須多行才有感）

- `content-start` / `content-end` / `content-center`
- `content-between` / `content-around` / `content-evenly`

**單一項目（align-self）：**

- `self-auto`（預設）
- `self-start` / `self-end` / `self-center` / `self-stretch`

```html
<div class="flex items-center">
  <div>Avatar</div>
  <div class="self-start">Title on top</div>
</div>
```

### 2.6 彈性大小（grow / shrink / basis）

- `flex-1`：`flex: 1 1 0%;` 常用「平均分配」
- `flex-auto`：`flex: 1 1 auto;`
- `flex-initial`：`flex: 0 1 auto;`
- `flex-none`：`flex: none;`（不縮不長）
- `grow-0` / `grow`：`flex-grow`
- `shrink-0` / `shrink`：`flex-shrink`
- `basis-*`：控制 `flex-basis`，如 `basis-1/2`、`basis-1/3`、`basis-full`

```html
<div class="flex gap-4">
  <aside class="w-64 flex-none">Sidebar</aside>
  <main class="flex-1">Content</main>
</div>
```

### 2.7 排序（order）

- `order-first` / `order-last`
- `order-none`（預設）
- `order-1` ~ `order-12`

```html
<div class="flex flex-col md:flex-row">
  <div class="order-2 md:order-1">Content</div>
  <div class="order-1 md:order-2">Sidebar</div>
</div>
```

### 2.8 Flex 常見版型範例

**導覽列：**

```html
<header class="flex items-center justify-between px-4 py-2 bg-slate-900 text-white">
  <div class="font-bold">Logo</div>
  <nav class="hidden md:flex gap-4">
    <a href="#">Home</a>
    <a href="#">About</a>
  </nav>
  <button class="md:hidden">☰</button>
</header>
```

**左右欄版型：**

```html
<section class="flex flex-col md:flex-row gap-4">
  <main class="flex-1">Main content</main>
  <aside class="w-full md:w-80 flex-none">Sidebar</aside>
</section>
```

## 3. Grid 版面（CSS Grid）

### 3.1 啟用 Grid

- `grid`：`display: grid;`
- `inline-grid`

```html
<div class="grid">...</div>
```

### 3.2 欄數（grid-template-columns）

- 固定欄數：`grid-cols-1` ~ `grid-cols-12`
- 移除定義：`grid-cols-none`

```html
<div class="grid grid-cols-1 md:grid-cols-3 gap-4">
  <article class="bg-white p-4 shadow">Card 1</article>
  <article class="bg-white p-4 shadow">Card 2</article>
  <article class="bg-white p-4 shadow">Card 3</article>
</div>
```

### 3.3 列數與自動列（rows）

- 固定列數：`grid-rows-1` ~ `grid-rows-6`、`grid-rows-none`
- 自動列高：
  - `auto-rows-auto`（預設）
  - `auto-rows-min` / `auto-rows-max`
  - `auto-rows-fr`

```html
<div class="grid grid-rows-3 auto-rows-min gap-2">
  <!-- 三列網格，每列最小高度 -->
</div>
```

### 3.4 欄/列間距（gap）

- `gap-*`：同時控制 row 與 column 間距
- `gap-x-*` / `gap-y-*`：單獨控制

常見：`gap-2`、`gap-4`、`gap-6`。

### 3.5 欄位跨越（col-span / row-span）

- `col-span-1` ~ `col-span-12`、`col-span-full`
- `row-span-1` ~ `row-span-6`、`row-span-full`
- 起訖位置：`col-start-*` / `col-end-*`、`row-start-*` / `row-end-*`

```html
<div class="grid grid-cols-4 gap-4">
  <div class="col-span-4 md:col-span-2">A</div>
  <div class="col-span-4 md:col-span-2">B</div>
  <div class="col-span-4">Full width footer</div>
</div>
```

### 3.6 內容流向（grid-auto-flow）

- `grid-flow-row`（預設）
- `grid-flow-col`
- `grid-flow-row-dense` / `grid-flow-col-dense`

```html
<div class="grid grid-cols-3 grid-flow-row-dense gap-2">
  <!-- 適合 Masonry 類似排法（有限制，但可稍微補洞） -->
</div>
```

### 3.7 Grid 對齊（整體與單一項目）

**整體對齊：**

- `justify-items-start|end|center|stretch`
- `items-start|end|center|stretch`（沿 block 軸）
- `place-items-center` 等（同時設定 justify + align）

**單一格對齊：**

- `justify-self-auto|start|end|center|stretch`
- `self-auto|start|end|center|stretch`
- `place-self-*`

```html
<div class="grid grid-cols-3 place-items-center">
  <div class="place-self-start">Left top</div>
  <div>Center</div>
</div>
```

### 3.8 Grid 常見版型範例

**卡片列表（RWD 三欄）：**

```html
<section class="grid gap-6 grid-cols-1 sm:grid-cols-2 lg:grid-cols-3">
  <article class="bg-white p-4 shadow">Card</article>
  <!-- ... -->
</section>
```

**內容 + 側欄：**

```html
<section class="grid gap-4 grid-cols-1 lg:grid-cols-[2fr,1fr]">
  <main>Main content</main>
  <aside>Sidebar</aside>
</section>
```

> 註：`grid-cols-[2fr,1fr]` 使用 arbitrary value 語法，需要 Tailwind v3+（3.4.7 支援）。

## 4. 切版常用輔助 Class（搭配 Flex / Grid）

### 4.1 版寬與置中

- `container`：內建 container，會依 breakpoint 改變寬度
- 固定/最大寬度：`w-full`、`max-w-md`、`max-w-lg`、`max-w-xl`、`max-w-2xl`、`max-w-4xl`、`max-w-7xl` 等
- 水平置中：`mx-auto`

```html
<div class="container mx-auto px-4">...</div>
```

### 4.2 高度與滿版

- `h-screen`：滿版高度
- `min-h-screen`：最小高度滿版
- `h-full`、`min-h-full` 等

```html
<section class="min-h-screen flex items-center justify-center">
  <div>Centered block</div>
</section>
```

### 4.3 顯示控制

- `block` / `inline-block` / `inline`
- `flex` / `inline-flex` / `grid` / `inline-grid`
- `hidden`
- 響應式顯示：`hidden md:block`、`flex md:hidden` 等

---

## 5. 小結與實戰建議

- **先選擇排版方式**：單方向排版用 Flex，多維度複雜版面用 Grid。
- **善用 gap**：`gap-*` 同時適用於 Flex 與 Grid，取代手動 margin 更乾淨。
- **RWD 由小到大**：先寫手機版 class，再用 `sm:`、`md:`、`lg:` 逐步調整。
- **盡量用語意化結構 + class**：HTML 結構乾淨、class 集中處理樣式，維護性更好。
