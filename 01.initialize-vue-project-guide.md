# Vue 創建新專案

> 開發現代前端(React, Vue, Angular, Svelte...)，需先安裝Node.js，本文以Vue為例

---

## 1. 先到官網查看指令

官方快速開始文件：[https://vuejs.org/guide/quick-start.html](https://vuejs.org/guide/quick-start.html)

```bash
npm create vue@latest
```

## 2. 選擇專案功能

```bash
? Project name: ... <your-project-name>
? Add TypeScript? ... No / Yes
? Add JSX Support? ... No / Yes
? Add Vue Router for Single Page Application development? ... No / Yes
? Add Pinia for state management? ... No / Yes
? Add Vitest for Unit Testing? ... No / Yes
? Add Cypress for End-to-End Testing? ... No / Yes
? Add ESLint for code quality? ... No / Yes
? Add Prettier for code formatting? ... No / Yes
```

![功能示意圖](image/step%202.png)

## 3. 安裝並啟動

```bash
cd <your-project-name>
npm install                     # 安裝依賴
npm run dev                     # 啟動開發伺服器
```

![啟動示意圖](image/step%203.png)

## 4. 專案結構

```bash
burn-in/
├── .editorconfig                # 編輯器配置
├── .gitattributes               # Git 屬性設定
├── .gitignore                   # Git 忽略檔案
├── .prettierrc.json             # Prettier 格式化配置
├── .vscode/                     # VS Code 設定
│   ├── extensions.json          # 推薦的擴充套件
│   └── settings.json            # 編輯器設定
├── public/                      # 靜態資源
│   └── favicon.ico              # 網站圖示
├── src/                         # 原始碼目錄
│   ├── router/                  # 路由配置
│   │   └── index.ts             # 路由定義
│   ├── stores/                  # Pinia 狀態管理
│   │   └── counter.ts           # 計數器 store 範例
│   ├── App.vue                  # 根組件
│   └── main.ts                  # 應用程式入口
├── env.d.ts                     # 環境變數型別定義
├── eslint.config.ts             # ESLint 配置
├── index.html                   # HTML 入口檔案
├── package.json                 # 專案依賴與腳本
├── README.md                    # 專案說明文件
├── tsconfig.app.json            # 應用程式 TypeScript 配置
├── tsconfig.json                # TypeScript 主配置
├── tsconfig.node.json           # Node.js TypeScript 配置
└── vite.config.ts               # Vite 建置配置
```
