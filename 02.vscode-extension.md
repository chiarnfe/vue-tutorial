# 推薦VS Code擴充套件

> 以下為目前專案常用的VS Code擴充套件

---

## 1. Vue Language Features (Volar)

官方文件：[https://marketplace.visualstudio.com/items?itemName=Vue.volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)

Volar 提供了 Vue 3 的語法高亮、智能感知、跳轉定義等功能，是 Vue 開發不可或缺的工具。

## 2. Prettier - Code formatter

官方文件：[https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)

Prettier 是一個代碼格式化工具，可以自動格式化代碼，保持代碼風格的一致性。

## 3. ESLint

官方文件：[https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)

ESLint 是一個代碼檢查工具，可以幫助我們發現代碼中的問題，提高代碼質量。

## 4. Mithril Emmet

官方文件：[https://marketplace.visualstudio.com/items?itemName=vscode-emmet/emmet](https://marketplace.visualstudio.com/items?itemName=vscode-emmet/emmet)

Emmet 是一個代碼片段工具，可以幫助我們快速生成代碼，特別是 HTML代碼。

## 5. Tailwind CSS Intellisense

官方文件：[https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)

Tailwind CSS Intellisense 提供了 Tailwind CSS 的智能感知功能，可以幫助我們快速編寫 Tailwind CSS 代碼。
