# vue-router 與 map.json 的設定說明

> vue-router是用於前端設定路由的套件，切換路由時，會自動載入相應的組件(component)。

---

## 1. vue-router 的基本設定

```ts
// src/routes/index.ts
import { createRouter, createWebHistory } from 'vue-router';

const routes = [
  {
    path: '/Default/Abnormal',
    component: () => import('../pages/EQPRAM/Abnormal.vue'),
  },
  {
    path: '/Default/DSEQPRAM',
    component: () => import('../pages/EQPRAM/DSEQPRAM.vue'),
  },
  {
    path: '/Default/TestEQPRAM',
    component: () => import('../pages/EQPRAM/TestEQPRAM.vue'),
  },
];

export default createRouter({
  scrollBehavior: () => ({ top: 0, left: 0 }),
  history: createWebHistory(),
  routes,
});
```

> 要特別注意`path`不能重複，否則只會出現第一個被match到的組件。

## 2. App.vue 中的設定

```vue
// src/App.vue
<template>
  <router-view />
</template>
```

> App.vue作為Vue的根組件，其中的router-view會被路由切換時，載入相應的組件(component)，平時不會更動這個組件。


## 2. map.json 的設定

`map.json` 用於設定路由與組件的對應關係，格式如下：

```json
{
  "專案名稱": [
    {
      "path": "路由",
      "component": "組件路徑"
    },
    ...
  ],
  ...
}
```

例如：
```json
{
  "APQP": [
    {
      "path": "/Default/Abnormal",
      "component": "../pages/EQPRAM/Abnormal.vue"
    },
    {
      "path": "/Default/DSEQPRAM",
      "component": "../pages/EQPRAM/DSEQPRAM.vue"
    },
    {
      "path": "/Default/TestEQPRAM",
      "component": "../pages/EQPRAM/TestEQPRAM.vue"
    }
  ],
  ...
}
```

> VITE_PROJECT 的值，就是 map.json 中的「專案名稱」，也必須是 map.json 中的key值， 才能正常啟動專案。
